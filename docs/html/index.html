construct¶

An extensible API for creative project and asset management.

Features¶

  • Manage Projects on your file system
  • Integrate with your Creative Software
  • Manage Workspaces and Publishes
  • Folder and Path Templating (fsfs and Lucidity)
  • Contextual API
  • Extensions, Actions, and Tasks

Core Packages¶

package description
construct_cpenv Cpenv Integration for env management
construct_launcher Application launcher
construct_maya Autodesk Maya integration
construct_nuke The Foundry Nuke integration
construct_ui Graphical user interface

Installation¶

$ pip install git+ssh://git@github.com/construct-org/construct.git
